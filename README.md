## 📌 Overview
This repository aims to provide:
- 📖 A structured list of Bangladeshi tech companies  
- 🔍 Easy navigation by **A–Z order**  
- 🛠️ Key details: Location, Tech Stack, Website, LinkedIn, Reviews  

---

## 🤝 Contribution Guide

We welcome all contributions! To add or update a company:  

1. **Fork** this repository  
2. **Add company info** in alphabetical order  
3. **Include the following details:**  

   - 🏢 Company name  
   - 📍 Location  
   - ⚙️ Technologies used  
   - 🌐 Official website  
   - 🔗 LinkedIn company page  
   - ⭐ Rating (if available on review sites)  

4. **Create a Pull Request** 🚀
